デフォルトのエディタを変更
==========================

2016年03月16日

(Ubuntu 15.10 i386)

.. code-block:: bash

   $ sudo update-alternatives --config editor

ちなみに、update-alternativesで設定されているグループがどれだけ
あるか知りたいときは

.. code-block:: bash

   $ update-alternatives --get-selections
